# Welcome to the KriStarter, React/Redux UI boilerplate

![welcome](https://media.giphy.com/media/26vUTlnHulTgAU7le/giphy.gif "Grill")

## Usage :

### With yarn :
```
git clone git@github.com:Nikulio/kristarter.git

cd kristarter

yarn
```

**for development:**

`yarn start`

**for production:**

`yarn build`

### With npm :
```
git clone git@github.com:Nikulio/kristarter.git

cd kristarter

npm i
```

**for development:**

`npm run start`

**for production:**

`npm run build`
